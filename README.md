# Документация, чтобы развернуть проект

## Внимание

Для запуска проекта, нужно иметь актуальную версию node.js: v22.17.0

## Клонирование репозитория и инициализация зависимостей

git clone https://github.com/distinkt-dd/shopinst

npm install

### или

yarn install

## Настройка переменных окружения

### Пояснение переменных окружения:

- VITE_API_ORIGIN=https://nti.urfu.ru
- VITE_API_KEY=api_key

## Разработка

npm run build

### или

yarn build

## Основные зависимости

- Vue 3 - фронтенд фреймворк с Composition API

- TypeScript - типизированный JavaScript

- Pinia - управление состоянием

- Vue Router 4 - маршрутизация

- Sass/SCSS - стилизация

- Vite - сборщик и dev-сервер
